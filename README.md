## 🚀 Tecnologias

Esse projeto foi desenvolvido com as seguintes tecnologias:

-   React.js
-   TypeScript
-   Next.js

## 💻 Projeto

O move.it é uma aplicação web desenvolvida com React.js no NLW4 para aplicação da técnica de Pomodoro com exercícios físicos básicos.
